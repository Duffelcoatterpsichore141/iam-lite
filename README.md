## 💻 What iam-lite does

iam-lite helps you manage user access in one place. It supports roles, rules, login flows, approval steps, and audit logs.

Use it to:

- give each user the right access
- keep role-based access under control
- set rule-based access for common cases
- connect sign-in with OAuth2 and OpenID Connect
- review access changes in an audit trail
- keep a record that helps with LGPD needs

## 🔐 Main features

### 👥 User and role control

Add users to roles and control what they can do. This keeps access clear and easy to manage.

### 🧩 Rule-based access

Set rules that decide access by user data, location, department, or other fields. This helps you handle more than simple roles.

### 🔑 OAuth2 and OIDC sign-in

Use standard login flows so users can sign in with supported identity systems.

### ✅ Approval flow

Send access changes through an approval step before they take effect.

### 🧾 Audit trail

Track who changed what and when. This helps with reviews and compliance checks.

### 🛡️ LGPD support

Keep a record of access and changes in a way that supports privacy control and data review needs.
